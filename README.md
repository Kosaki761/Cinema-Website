How to run :
1. Install Laragon
2. Open Laragon and add CinemaBooking folder to 'root'.
3. Open database on laragon and create new database named cinema.
4. Run the SQL query.

*NEED TO MANUALLY ADD THE DATA ON DATABASE OR ADMIN PAGE*
